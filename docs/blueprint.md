# **App Name**: FastFoodGo

## Core Features:

- Product Catalog: Display a catalog of food items with images and descriptions.
- Order Placement: Allow customers to add items to their cart, specify quantities, and calculate the total cost.
- Suggested Products: AI-powered feature to suggest additional items based on current cart contents and order history using a tool that weighs prices and ingredient compatibility. 
- Payment Options: Offer payment gateway integration and cash on delivery options.
- Order Management (Admin): Admin interface for adding products, managing inventory, and viewing sales statistics (monthly and annual).
- Order Notifications: Real-time notifications for new orders to be sent to the kitchen.
- Delivery Tracking: Map integration to allow customers to choose delivery address and track their order status, with separate driver view for delivery assignments.

## Style Guidelines:

- Primary color: Vibrant red (#F44336) to convey energy and appetite.
- Background color: Light grey (#EEEEEE), a desaturated red, to give emphasis to the vibrant primary color.
- Accent color: Yellow-orange (#FF9800), analogous to the primary, to draw attention to calls to action.
- Body and headline font: 'PT Sans' for a modern and accessible feel.
- Use food-related icons from a simple, outlined set for clarity.
- Card-based layout for products, making browsing intuitive.
- Subtle transitions when adding items to cart or updating order status.
